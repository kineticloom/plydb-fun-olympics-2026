# Contributing

## Contributing an analysis

Have an interesting angle on the data? Contributions are welcome.

Add your writeup as a Markdown file in the [/analysis/](/analysis/) folder and
open a pull request. Good analyses tend to:

- Start with a clear question or hypothesis
- Show your work (include the queries or steps used to reach your conclusions)
- Call out anything surprising or counterintuitive in the data

## Found an error?

The Olympics data and AI-generated analyses in this repo may contain mistakes —
misattributed medals, bad joins, or just a hallucination that slipped through.
If you spot something wrong, please
[open an issue](https://github.com/kineticloom/plydb-fun-olympics-2026/issues)
with a description of the error and, if possible, the correct value or source.
